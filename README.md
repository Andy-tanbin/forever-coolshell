## 使用方法

下载 Release 页面的执行文件，或使用 Docker 一键启动容器即可。

网站电子存档默认使用 `8080` 端口，如需改变端口，请指定环境变量 `PORT`，如：

```bash
PORT=10240 ./coolshell
```
